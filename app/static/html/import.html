📥 Import Data

Step 1 — Upload Excel File

আপনার পুরনো monthly Excel ফাইল (Date, Details, Remarks, IN, OUT কলাম সহ) আপলোড করুন। Upload করার পর প্রতিটা row preview এ দেখাবে — তারপর আপনি বেছে নিতে পারবেন কোনগুলো import হবে।

Step 2 — Review & Edit Rows
Date Type Category Note Amount Account Action
⚠️ খেয়াল রাখুন:
• TOTAL / header row গুলো automatically skip হবে।
• একই row দুইবার import করলে transaction duplicate হয়ে যাবে।
• শুধু checked (✓) row গুলোই database এ save হবে।
• Transfer type select করলে — From এবং To দুটো account select করতে হবে (যেমন Cash থেকে Bank, বা Bank থেকে Cash)। Transfer এ Category লাগে না।
